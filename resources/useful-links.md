## Loot Filtr

Loot filter ukrywa lub wyróżnia loot na podstawie reguł — bardzo zalecany!

- **Filterblade.xyz** — najpopularniejsza strona do tworzenia filtrów
- Pobierz gotowy filtr od "NeverSink" — jeden z najpopularniejszych twórców filtrów
- Wgraj filtr przez: Options → UI → Loot Filter

### Poziomy filtrów
| Poziom | Dla kogo |
|--------|----------|
| Soft / Regular | Nowi gracze — pokazuje prawie wszystko |
| Semi-strict | Kampania → wczesny endgame |
| Strict | Mid endgame |
| Very Strict | Late endgame |
| Uber Strict | Pinnacle farmowanie |
