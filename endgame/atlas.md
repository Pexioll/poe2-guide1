# Atlas of Worlds — Przewodnik

---

## Czym jest Atlas?

**Atlas of Worlds** to sieć połączonych map (Waystones), przez którą postępujesz w endgame. Każda mapa to odrębna lokacja z wrogami, potworami i bossem.

---

## Waystones (Klucze do Map)

- **Waystone** = klucz do mapy; zużywany po użyciu
- Tiery: **T1** (najłatwiejsze) do **T16+** (najtrudniejsze)
- Wyższy tier = trudniejsze wrogowie + lepszy loot
- Zdobywasz Waystones z:
  - Lootu z map
  - Vendorów (craft)
  - Handlu z graczami

### Ulepszanie Waystones

```
Normal Waystone → [Transmutation Orb] → Magic Waystone
Magic Waystone → [Alteration Orb] → zmień mody (Magic)
Magic Waystone → [Regal Orb] → Rare Waystone
Rare Waystone → [Exalted Orb] → dodaj mod (Rare)
Rare Waystone → [Chaos Orb] → zmień wszystkie mody (Rare)
```

### Mody Waystones

| Rodzaj Modu | Efekt |
|-------------|-------|
| Quantity (IIQ) | Więcej lootu dropuje |
| Rarity (IIR) | Lepsza jakość lootu |
| Pack Size | Więcej wrogów w packach |
| Enemy Damage | Wrogowie zadają więcej obrażeń |
| Enemy Life | Wrogowie mają więcej HP |
| Elemental Penetration | Niebezpieczne — wrogowie ignorują część twoich res |
| Curse | Wrogowie nakładają przekleństwa |

> **Zasada:** Im więcej "złych" modów, tym więcej lootu — ale ważne jest przeżycie!

## Atlas Passive Tree

Oddzielne drzewko pasywne dedykowane endgame. Punkty zdobywasz przez:
- **Completing Waystones** — pierwsza ekranizacja mapy
- **Bossowie map**

### Strategie Atlas Passive Tree

#### Strategia Generalista (dla nowych)
- Inwestuj w **bonusy do wszystkich map** (Quantity, Pack Size)
- Nie skupiaj się za bardzo na jednej mechanice

#### Strategia Specjalizacja (dla zaawansowanych)
- Wybierz jedną lub dwie league mechaniki i zainwestuj głęboko
- Przykład: Breach farm → weź wszystkie Breach nodes
- Wyższe returns per hour, mniejsza różnorodność

#### Popularne Focus Punkty
| Focus | Co to daje |
|-------|-----------|
| Breach | Więcej Breach + Breachstones |
| Ritual | Więcej Ritual + Omens |
| Delirium | Więcej Delirium + Simulacrum |
| Expedition | Więcej logbooks, Remnants |
| Bossy | Więcej boss encounters |
| Currency | Bezpośredni drop rate bonus |

## Nawigacja po Atlasie

- **Mapa Atlasu** — otwierasz przez UI
- Połączone mapy — po ukończeniu mapy otwierają się sąsiednie
- **Kishtvar/Strongholds** — specjalne punkty na mapie
- Boss Rooms — specjalne mapy z bossami

### Progression Atlas
```
Tier 1-5 → Odkryj podstawową siatkę
Tier 6-10 → Zacznij wchodzić w środkowy atlas
Tier 11-15 → Outer ring atlasu
T16+ → Pinnacle content
```

## Map Mods — Niebezpieczne Mody

Unikaj (lub bądź ostrożny) jeśli twój build jest słaby na dane mody:

| Mod | Dlaczego niebezpieczny |
|-----|----------------------|
| "Monsters reflect X% of Elemental Damage" | Własne obrażenia cię zabijają — sprawdź czy build jest immune |
| "Players cannot Leech" | Buildy leech-based tracą sustain |
| "Cannot Regenerate Life/ES" | Buildy regen-based tracą sustain |
| "Temporal Chains" | Spowalnia Flask użycie i ruch |
| "+X% Monster Elemental Penetration" | Twoje res efektywnie spadają |
| "Monsters deal Chaos Damage" | Niebezpieczne bez Chaos res |

## Tiering Waystones — Efektywność

Ogólna zasada: **nie dawaj się za mocno ulepszonym mapom jeśli umierasz**.

Koszt śmierci w mapie:
- Tracisz XP (penalty)
- Możesz stracić map Waystone jeśli się cofniesz
- Stracsz czas

**Lepiej:** Wolniejsze czyste mapy niż szybkie z ryzykiem śmierci (szczególnie w endgame).
