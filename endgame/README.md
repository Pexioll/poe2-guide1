## Szybki Start — Co Robić Po Kampanii

1. **Otwórz atlas** — znajdź niskie tiery map (T1–T5) i czyść je
2. **Ulepsz Waystones** przez currency (Transmutation → Alteration → Regal → Exalt)
3. **Inwestuj w Atlas Passive Tree** — wybierz swój focus
4. **Szukaj league mechanic** w mapach — to główne źródło waluty
5. **Handluj** — brakujące itemy kup, nadmiarowe sprzedaj
6. **Skaluj się w górę** — T10 → T15 → T16 → Pinnacle
