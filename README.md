# poe2-guide

Notatki do szybkiego przejscia kampanii w Path of Exile 2, z naciskiem na leaguestart Mercenary Grenades.

## GitHub Pages

Repo jest gotowe do hostowania jako strona przez GitHub Pages.

### Jak uruchomic

1. Wrzuc repo na GitHub.
2. Wejdz w `Settings -> Pages`.
3. W `Build and deployment` ustaw:
	- Source: `Deploy from a branch`
	- Branch: `main`
	- Folder: `/ (root)`
4. Zapisz i poczekaj okolo 1-2 min.

Strona bedzie pod adresem:

`https://twoj-login.github.io/poe2-guide/`

Strona startowa jest w pliku `index.md`.

## Co jest w projekcie

Projekt ma aktualnie 3 glowne obszary:

1. campaign-default
Sciezka kampanii krok po kroku, podzielona na akty i interludes.

2. League start/build
Skondensowany build summary pod Mercenary Grenades (rotacja, gem sety, priorytety supportow, gear).

3. League start/items
Progresja baz crossbow oraz praktyczny crafting checkpoint po checkpointcie.

## Jak tego uzywac podczas runa

1. Otworz taski aktualnego aktu
Korzystaj z checklisty w campaign-default i odhaczaj kroki po kolei.

2. Trzymaj obok build summary
W League start/build/mercenary-grenades.md masz rotacje i gem setupy na przedzialy leveli:
- lvl 1-13
- lvl 14-51
- lvl 52+

3. Pilnuj breakpointow weapon/craft
W League start/items/crossbows bases + crafting.md sa kluczowe momenty:
- lvl 10: Sturdy Crossbow
- lvl 16: Varnished Crossbow (glowny craft)
- lvl 33: Bombard Crossbow
- Act 4/Interludes: Cannonade Crossbow + Greater Runes

4. Priorytet praktyczny
- najpierw kampania (tasks)
- rownolegle gemy i rotacja (build)
- na checkpointach i przy vendorach upgrade broni (items)

## Nawigacja na stronie

- Campaign: Act 1-4 + Interludes
- League Start: build + crossbow crafting
