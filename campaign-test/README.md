## Format pliku

Kazdy plik aktu ma takie bloki:

```
### Nazwa strefy
- [ ] Krok 1
- [ ] Krok 2 (OPT)
> NAGRODA: [co sie dostaje]
> GEM CHECK: [UZUPELNIJ - co tutaj wziac]
> VENDOR CHECK: [UZUPELNIJ - czego szukac]
```

Bloki oznaczone `> GEM CHECK` i `> VENDOR CHECK` wypelniasz sam pod swoj build.
Bloki `> NAGRODA` sa juz wypelnione na podstawie poradnika.

## Legenda

- [ ] = do zrobienia
- [x] = zrobione (zaznaczasz podczas gry)
- (OPT) = opcjonalne, oplacalne ale nie konieczne
- NAGRODA = nagroda z tego kroku
- GEM CHECK = miejsce na Twoje notatki o gemach
- VENDOR CHECK = co sprawdzic u vendora
