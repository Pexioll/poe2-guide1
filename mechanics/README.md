## Najważniejsze różnice PoE2 vs PoE1

| Mechanika | PoE1 | PoE2 |
|-----------|------|------|
| Gemy | Wsadzane w sockety itemów | Są osobnymi itemami |
| Dodge | Nie było | Inherentny dla wszystkich klas |
| Stagger/Break | Brak | Nowa mechanika |
| Flask | Naboje | Cooldown-based |
| Ascendancy | Po labyrinthach | Po Trials |
| Mana | Klasyczny system | Bardziej restrykcyjny |
| Miniony | Do 20+ | Mniejsze limity |
