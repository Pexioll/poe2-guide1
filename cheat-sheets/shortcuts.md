# Cheat Sheet — Skróty Klawiszowe

> Domyślne skróty — możesz je zmienić w Ustawieniach → Keybinds.

---

## Podstawowe

| Akcja | Skrót |
|-------|-------|
| Otwórz Menu | Escape |
| Otwórz Stash | E (przy skrzyni) |
| Otwórz Ekwipunek | I |
| Otwórz Passive Tree | P |
| Otwórz Skill Panel | S |
| Otwórz Atlas | G |
| Otwórz Party | O |
| Chat | Enter |
| Automapa (toggle) | Tab |
| Powiększ automapę | Ctrl+Tab |

---

## Walka

| Akcja | Skrót |
|-------|-------|
| Dodge Roll | Spacja |
| Skill 1 | Q |
| Skill 2 | W |
| Skill 3 | E |
| Skill 4 | R |
| Skill 5 | T |
| Potion 1–5 | 1, 2, 3, 4, 5 |
| Stój i atakuj | Shift + LPM |
| Atakuj w kierunku | A + LPM |

---

## Przedmioty i Handel

| Akcja | Skrót |
|-------|-------|
| Przenieś do Stash (z ekw.) | Ctrl + klik |
| Podziel stack | Shift + klik |
| Sprawdź ceny (jeśli masz tool) | Ctrl + D |
| Identyfikuj item | PPM (Scroll of Wisdom) |
| Hover + Alt | Pokaż zaawansowane info o itemie |
| Hover + Shift | Porównaj z wyposażonym |
| Tag for Trade | PPM → Tag for Trade |

---

## Zaawansowane

| Akcja | Skrót |
|-------|-------|
| Kopiuj opis itemu | Ctrl + C (hover) |
| Portal Scroll | TP Scroll z ekwipunku |
| Social Panel | S lub dedykowany klawisz |
| Guild | dedykowany klawisz |
| Wiadomość prywatna | @NickGracza |
| Trade Chat | /trade (typ) |
| Global Chat | /global |

---

## Modyfikatory Kliknięcia

| Combo | Efekt |
|-------|-------|
| Ctrl + klik | Przenieś item (stash ↔ ekwipunek) |
| Shift + klik | Podziel stos currency |
| Alt + klik | Sortuj lub inne (zależy od kontekstu) |

---

## Komendy Chat

| Komenda | Efekt |
|---------|-------|
| `/help` | Lista komend |
| `/hideout` | Teleport do hideout |
| `/dnd [message]` | Do Not Disturb mode |
| `/afk [message]` | Away from keyboard |
| `/invite [nick]` | Zaproś do party |
| `/kick [nick]` | Wyrzuć z party |
| `/whois [nick]` | Informacje o graczu |
